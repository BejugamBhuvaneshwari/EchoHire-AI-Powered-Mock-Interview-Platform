## 📖 Overview

EchoHire is an AI-powered mock interview platform that helps users prepare for technical and HR interviews through realistic voice-based conversations.

The application generates interview questions using **Google Gemini AI**, conducts interviews using **Vapi AI Voice Agents**, and provides detailed AI-generated feedback to help users improve their communication, technical knowledge, confidence, and overall interview performance.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/EchoHire.git

cd EchoHire
```

Install dependencies

```bash
npm install
```

---

## 🔑 Environment Variables

Create a `.env.local` file in the project root.

```env
NEXT_PUBLIC_VAPI_WEB_TOKEN=

NEXT_PUBLIC_VAPI_WORKFLOW_ID=

GOOGLE_GENERATIVE_AI_API_KEY=

NEXT_PUBLIC_BASE_URL=

NEXT_PUBLIC_FIREBASE_API_KEY=

NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=

NEXT_PUBLIC_FIREBASE_PROJECT_ID=

NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=

NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=

NEXT_PUBLIC_FIREBASE_APP_ID=

FIREBASE_PROJECT_ID=

FIREBASE_CLIENT_EMAIL=

FIREBASE_PRIVATE_KEY=
```
